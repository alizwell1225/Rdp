## Features Visible in UI

### Top Section
- **Title Bar**: "gRPC Configuration"
- **Mode Label**: Displays current mode (Client/Server) with color coding

### Connection Settings Group
- **Host Field**: Text input for server address
  - Default: "localhost"
  - Examples: "192.168.1.100", "myserver.com"
  
- **Port Field**: Numeric input with up/down arrows
  - Range: 1 to 65535
  - Default: 50051

### File Paths Group
- **Log File Path**: 
  - Text input for log file location
  - Browse button (...) opens SaveFileDialog
  - Example: `C:\App\Log\grpc.log`

- **Storage Root / Path**:
  - Text input for storage directory
  - Browse button (...) opens FolderBrowserDialog
  - Label changes based on mode:
    - Client: "Storage Root / Path" (for downloads)
    - Server: "Storage Root / Path" (for uploads)

### Action Buttons
- **Load Config...**: Opens file dialog to load existing JSON config
- **Reset Defaults**: Resets all fields to default values (with confirmation)
- **Save**: Validates inputs and saves to JSON file
- **Cancel**: Closes dialog without saving changes

## Color Coding

### Mode Indicator
- **Client Mode**: `Mode: Client` in **Dark Green** (#006400)
- **Server Mode**: `Mode: Server` in **Dark Blue** (#00008B)

## Usage in Application

The form can be opened from:
1. **Menu**: File â†’ Settings â†’ gRPC Configuration
2. **Button**: "Settings" or "Config" button in main form
3. **Toolbar**: Settings icon in application toolbar
4. **Context Menu**: Right-click â†’ Configuration

## Form Properties

- **Size**: 484 x 298 pixels
- **Border Style**: FixedDialog (non-resizable)
- **Start Position**: CenterParent
- **Minimize/Maximize**: Disabled
- **Modal**: Yes (blocks parent form until closed)

## Keyboard Shortcuts

- **Tab**: Navigate between fields
- **Enter**: Save (when Save button has focus)
- **Escape**: Cancel
- **Alt+L**: Focus Load Config button
- **Alt+R**: Focus Reset Defaults button
- **Alt+S**: Focus Save button
- **Alt+C**: Focus Cancel button
